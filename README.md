# Сдача лабораторных работ.
Лабораторные работы выполнены соответствующих их номерам папках проекта.
## Описание проекта:
Необходимо спроектировать и реализовать программную систему для автоматизированного управления доступом пользователей к распределённым удалённым серверам, предназначенным для маршрутизации пользовательского сетевого трафика через внешние узлы.

В рамках проекта требуется разработать архитектуру системы, которая будет обеспечивать:

- автоматическую выдачу доступа пользователям без участия администратора;
- хранение информации о выданных доступах, сроках их действия и привязке к пользователям;
- автоматическое удаление доступа по истечению срока действия;
- продление доступа при наличии активной подписки и успешного проведения платежа;
- интеграцию с платёжными сервисами для приёма разовых и рекуррентных платежей;
- поддержку тарифных планов с различными условиями доступа;
- сбор и хранение статистики использования серверов (активные подключения, объём переданных данных, нагрузка);
- равномерное распределение пользователей между удалёнными серверами с учётом их текущей загрузки;
- возможность динамического добавления и отключения серверов без остановки системы;
- централизованное администрирование серверов и пользовательских доступов;
- логирование действий пользователей и административных операций.

Проектируемая система должна быть рассчитана на одновременную работу более 1000 пользователей, обеспечивать стабильную работу при высокой нагрузке, горизонтальное масштабирование компонентов и отказоустойчивость при сбоях отдельных узлов.

Архитектура системы должна минимизировать ручное администрирование, обеспечивать безопасность передаваемых данных и предусматривать возможность дальнейшего развития функциональности без существенной переработки базовых компонентов.
